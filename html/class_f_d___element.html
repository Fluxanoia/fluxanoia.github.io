Constructs a FD_Element.

Note, if the scale_y is less than zero, then it will use the scale parameter as the vertical and horizontal scale. By default, scale_y is less than zero.

Parameters
imageThe image to be used.
xThe x-coordinate of the element.
yThe y-coordinate of the element.
angleThe angle of the element.
zThe layer of the element.
scaleThe horizontal scale for the element.
scale_yThe vertical scale for the element.
camera_boundWhether the element should be drawn relative to the camera.
styleThe drawing style for the element (see FD_DrawStyle).
