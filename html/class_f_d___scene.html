◆ generateObjectList()

FD_ObjListIndex FD_Scene::generateObjectList ( )

Generates a new index for registering instances of FD_ObjectGroup.

The FD_Scene can only draw a single object list at a time, meaning instances of FD_State (or anything really) shpuld have their own (or many) object lists that they will register their instances of FD_ObjectGroup to for drawing.

Returns
The index for the new object list.
