void FD_Line::supplyPoints ( SDL_Point *  p1,
SDL_Point *  p2 
)

Supplies pointers to points to the line.

The line will use the value of the supplied points whenever possible. Otherwise, it will fallback to the last given value. The supplied points can be wiped with removePoints.
