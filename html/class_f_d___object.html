Detailed Description

The virtual FD_Object class, the framework for the individual elements drawn by the engine.

Constructs a FD_Object.

Note that this class is simply a framework, and whilst some of these variables may not make sense for certain implementations of FD_Object - their core meaning should be maintained where possible.

For example, there's no general notion of the 'width' of a line, but the FD_Line class maintains the width as the absolute distance between its points in the x-axis. Similarly, the angle is maintained at zero.
