void FD_Box::supplyRect ( SDL_Rect *  rect)

Supplies a pointer to a rectangle to the box.

The box will use the values of the supplied rectangle whenever possible. Otherwise, it will fallback to the last given values. The supplied rectangle can be wiped with removeRect.
