Detailed Description

The FD_SFX class, containing and managing a SFX chunk.

Constructs a FD_SFX.

SFX chunks are retrieved in a unique way through the use of values. If the value is zero, the class will directly use the path given by the registry. However, if the value is non-zero, it will append it to the end of the path.

For example, you can label similar sounds: "blip1.wav", "blip2.wav", etc. and add "blip.wav" to the registry and construct the FD_SFX class with values 1, 2, etc.

Warning
This only works for .wav files.

registryThe registry to retrieve the SFX path from.
