Detailed Description

The class that can be controller by FD_Looper.

This class should be inherited by other classes to allow them to be used within the FD_Looper class. FD_Looper calls update at a fixed, given rate, render as much as possible and, pushEvent as events occur. This class is of little use otherwise.
