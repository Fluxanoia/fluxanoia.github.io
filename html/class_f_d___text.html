Detailed Description

The FD_Text class, allowing for text image manipulation.

Constructs a FD_Text.

This class consists of three text elements, the prefix, main text and, suffix. The prefix and suffix are input on construction and never change. The main text is input on construction but can be changed at anytime with changeText - reproducing the texture again with the new main body.

This system is useful for displaying scores, the prefix could be "You have " and the suffix could be " points.".

Parameters
rendererThe renderer used to produce the text texture.
fontThe font used by the text.
prefixThe (fixed) prefix of the text.

suffixThe (fixed) suffix of the text.
colourThe colour of the text.
xThe x-coordinate of the text.
yThe y-coordinate of the text.
