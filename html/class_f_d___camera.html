Detailed Description

The FD_Camera class, allowing for abstract coordinate systems.

The FD_Camera class works with FD_ObjectGroup and FD_Scene to translate and render images. FD_Camera does not draw itself, it manipulates destination rectangles to allow objects to be places appropriately.
